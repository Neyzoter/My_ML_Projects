# 说明
该文件包含卷积神经网络算法。

输入参数为(m,x,y,ch)，即（图像个数，x轴像素个数，y轴像素个数，通道数）。

# 系统说明

作用：卷积神经网络识别笑脸。

模型：卷积神经网络

策略：交叉熵损失函数-1/n(ylog(a)-(1-y)log(a))

算法：Adam

备注：训练集不够大，当对自己的照片进行识别时，效果不佳。

Train loss =  0.0120

Train accuracy =  0.9983

Test  loss =  0.102046815952

Test accuracy =  0.980000003974
